# Bot Protetor de Golpes — Telegram MVP

Bot acadêmico para análise de golpes digitais e fake news voltado para idosos.

## Funcionalidades
- análise de texto
- análise de imagem via OCR
- cadastro de Anjo da Guarda
- alerta automático em casos críticos

## Comandos
/start
/meu_id
/anjo ID_DO_FAMILIAR
